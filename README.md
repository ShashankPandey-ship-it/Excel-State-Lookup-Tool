# 📍 CSJ Sales State Lookup Tool – Excel Project

## 📊 Project Overview

The **CSJ Sales State Lookup Tool** is an interactive Excel-based dashboard designed to help users quickly select U.S. states and view relevant location details.
The tool focuses on improving usability, navigation, and lookup efficiency through a structured visual interface.

---

## 🎯 Project Objective

* Build an interactive Excel UI for state selection
* Enable quick navigation to state-level details
* Demonstrate Excel dashboard design and user experience skills
* Support sales or regional data lookup scenarios

---

## 🛠 Tools & Technologies Used

* Microsoft Excel
* Interactive Shapes & Buttons
* Sheet Linking & Navigation
* Lookup Functions (such as VLOOKUP / INDEX-MATCH)
* Dashboard Layout & Formatting

---

## 📌 Key Features

* Clickable state selection panel
* Dedicated **Details button** for navigation
* Organized grid layout for easy state identification
* Clean corporate-themed dashboard design
* Supports regional lookup for business use cases

---

## 💼 Business Use Case

This tool can be used in:

* Sales territory management
* Regional performance tracking
* Location-based customer analysis
* Administrative lookup systems
* Operational dashboards
